## Our SIH 2025 Project: **KRISHI HEDGE**

A **virtual hedging & price risk management platform** built for India‚Äôs 50M+ oilseed farmers.

### Key Features

* **AI-Based Price Forecasting** (Prophet + ARIMA)
* **Blockchain-Backed e-Contracts** (IPFS + Polygon)
* **Virtual Futures Simulation / Hedging Sandbox**
* **Mobile-First, Offline-Capable Architecture**
* **Farmer Education Modules (Financial Literacy)**
* **NCDEX-Style Transparent Mock Orderbook**
* **Real-Time Alerts (Push / SMS)**
* **FPO Integration for Collective Hedging**

---

## Tech Stack

**Frontend:** React Native, Next.js, NativeWind
**Backend:** Node.js, FastAPI, TypeScript
**ML Models:** Prophet, ARIMA, Statsmodels
**Database:** Supabase
**Blockchain:** IPFS, Polygon (Testnet/Mainnet-ready)
**Security:** SHA-256 hashing, role-based authentication
